# Workspace Configs

This folder contains optional workspace files for different ecosystems.
Use only what you need:

- Nx (JS/TS): `nx.json`, `workspace.json`
- Turborepo (JS/TS): `turbo.json`, `package.json`
- Cargo workspace (Rust): `Cargo.toml`
- Poetry workspace (Python): `pyproject.toml`
- Optional polyglot descriptor: `workspace.yaml` (for custom tooling)

You can keep unused files, but it’s cleaner to delete what you don’t use.
