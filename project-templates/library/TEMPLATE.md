# Library / SDK Template

A skeleton for reusable libraries and SDKs.

## Recommended docs profile
- `lib` (enables `docs/design/` only)

## What this template provides
- modular library layout in `src/`
- tests split: unit + integration
- `examples/` as living documentation

## Notes
- Keep public API stable and versioned (SemVer).
- Document breaking changes and update CHANGELOG on releases.
